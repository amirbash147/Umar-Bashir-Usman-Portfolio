Project Two: Personal Blog Layout

HTML CSS Layout

In this project, I practiced building a homepage layout for a simple personal blog. It includes a header, a navigation bar, a list of sample blog post titles, and a footer, all styled using CSS to look organized and easy to read.

What I learned from this project:

  • How to organize a page into header, main content, and footer sections.
  • How to style lists of content such as blog post titles.
  • How to use colors and spacing to separate different parts of a page.
