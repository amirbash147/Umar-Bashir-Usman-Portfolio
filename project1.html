Project One: Survey Form Page

HTML CSS Forms

This project was one of the first practice pages I built on freeCodeCamp. The goal was to design a page that looks like a survey form, using basic HTML form elements such as text inputs, radio buttons, and checkboxes, then styling them with CSS.

What I learned from this project:

  • How to structure a form using HTML.
  • How to style labels, inputs, and buttons with CSS.
  • How to use padding and margin to make a form look neat.
